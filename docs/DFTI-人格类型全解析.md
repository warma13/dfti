# DFTI 三角洲行动人格测试 - 结果分析与评语

> 测测你是哪种战场人格？四维度十六型，找到你的干员灵魂

---

## 四维度说明

| 维度 | 左极 | 右极 | 核心问题 |
|------|------|------|----------|
| **A/D** | 猛攻 (Attack) | 防守 (Defend) | 你倾向进攻还是防守？ |
| **T/L** | 团队 (Team) | 独狼 (Lone) | 你依赖团队还是单干？ |
| **C/R** | 战斗 (Combat) | 资源 (Resource) | 你追求击杀还是搜刮？ |
| **S/I** | 策略 (Strategic) | 直觉 (Instinct) | 你靠计划还是靠本能？ |

每个维度根据答题得分判定方向，四个维度组合出 2x2x2x2 = **16 种人格类型**。

---

## 猛攻 (A) 系列 - 8 种类型

### ATCS - 远见的鹰

- **关键词**: 进攻指挥 / 信息碾压 / 团队大脑 / 全局掌控
- **推荐干员**: 露娜 / 银翼

> 你是那种开局就喊「听我指挥」的人，关键是队友还真的会听。你擅长在进攻中保持清醒，一边架枪一边报点，脑子里永远有一张实时更新的战场地图。你不是最能打的，但你在的队伍莫名其妙就是赢。别人打的是枪法，你打的是信息差。

---

### ATCI - 善战的狼

- **关键词**: 狼群冲锋 / 兄弟同心 / 正面刚枪 / 血性压制
- **推荐干员**: 红狼 / 威龙

> 你和兄弟们就是一群饿狼。开局不废话，卡好点位直接冲，前排倒了后排补，打的就是配合和血性。你不需要什么精密战术，因为你的战术就是——「兄弟们跟我上，莽就完了」。红狼冲锋的时候你第一个跟上，威龙虎蹲炮的时候你第一个补枪。

---

### ATRS - 奉献的鹤

- **关键词**: 进攻奶妈 / 烟雾大师 / 无私续航 / 幕后英雄
- **推荐干员**: 蜂医 / 蝶

> 你是那个队友血量永远满的原因。进攻的时候你不抢人头，但你的烟雾弹永远扔在最关键的位置，你的激素枪永远对准最需要的人。你把「打辅助」变成了一门艺术——队友以为自己是靠操作赢的，其实是你在后面续了三管血。

---

### ATRI - 热血的金毛

- **关键词**: 万金油 / 热心过头 / 义气冲天 / 有你真好
- **推荐干员**: 乌鲁鲁 / 蜂医

> 你是队伍里啥都干的那个人——需要冲锋你冲锋，需要拉人你拉人，需要搜物资你也搜。你不一定每件事都做得最好，但你永远是最积极的那个。你凭直觉判断队伍现在最缺什么，然后立刻去补。虽然偶尔会因为太热心冲过头，但队友都知道你是好人。

---

### ALCS - 孤傲的豹

- **关键词**: 精准猎杀 / 冷酷高效 / 独狼战术 / 来去无踪
- **推荐干员**: 骇爪 / 无名

> 你是最危险的独行者。不报点不开麦，但击杀播报上你的名字出现得最频繁。你提前算好每个路口的时机，精准到秒地完成猎杀然后消失。队友可能全程都不知道你在哪，但对面全程都在问「那个人在哪」。你像骇爪一样无声，像无名一样致命。

---

### ALCI - 威风的龙

- **关键词**: 一人成军 / 绝境翻盘 / 操作怪物 / 天生战神
- **推荐干员**: 威龙 / 无名

> 你就是那个让对面五个人都不敢推的存在。1v3是热身，1v5是日常，你不需要队友是因为队友跟不上你的节奏。你凭本能打枪，凭天赋读局，凭一口气翻盘。你是威龙的灵魂——动力推进冲进去，虎蹲炮砸下来，磁吸炸弹贴上去，对面直接碎。

---

### ALRS - 狡猾的狐

- **关键词**: 闷声发财 / 路线规划 / 风险管控 / 效率至上
- **推荐干员**: 骇爪 / 比特

> 别人在打仗，你在发财。你是跑刀界的精算师，哪个房间刷什么物资、哪条路线最安全、怎么避开交火区，你比地图设计师还清楚。你独来独往不是因为社恐，是因为带队友会拖慢你的搜刮效率。你的背包永远是满的，你的KD永远是谜。像骇爪一样灵巧，像比特一样会算。

---

### ALRI - 随性的猫

- **关键词**: 随心所欲 / 佛系玩家 / 独自漫游 / 享受过程
- **推荐干员**: 蛊 / 无名

> 你打三角洲的方式就像猫逛后院——走到哪算哪，看到什么摸什么。你不追求击杀数也不执着搜刮量，偶尔遇到个敌人打一架，偶尔捡到个好东西开心一下。你是唯一一个在烽火地带还能走出散步感的人。队友急得跳脚的时候，你在天台看日落。

---

## 防守 (D) 系列 - 8 种类型

### DTCS - 沉稳的熊

- **关键词**: 据点堡垒 / 防守指挥 / 稳如磐石 / 绞肉机
- **推荐干员**: 深蓝 / 牧羊人

> 你是据点防守的教科书。每个路口架谁、每扇门朝哪开、烟雾弹什么时候扔，你安排得比赛伊德守大坝还严密。你蹲在掩体后面像一座山，敌人推进来才发现这个点位是个绞肉机。你不追求华丽操作，你追求的是——「来一个死一个，来一队灭一队」。

---

### DTCI - 阴险的蛇

- **关键词**: 陷阱大师 / 阴险布局 / 出其不意 / 心理战专家
- **推荐干员**: 牧羊人 / 比特

> 你不是在防守，你是在设局。每个角落都是你的陷阱，每扇门后面都可能有你的「惊喜」。你最喜欢的战术就是让队友把人往你这边赶，然后在敌人以为安全的时候一口咬上去。牧羊人的小面包、比特的蜘蛛巢，都是你的毒牙。对面打完一局只想说一个字：阴。

---

### DTRS - 深谋的鸮

- **关键词**: 后方军师 / 资源调度 / 情报掌控 / 隐形MVP
- **推荐干员**: 蛊 / 银翼

> 你在后方运筹帷幄，前线的兄弟们不知道为什么弹药永远够用、受伤总能及时回血、情报总是恰好送到。你像德穆兰一样冷静地计算每一份资源的分配，只不过你站在正义的一边。你不出现在击杀榜上，但每一场胜利都有你的影子。

---

### DTRI - 贪吃的猪

- **关键词**: 搜刮狂魔 / 物资嗅觉 / 快乐仓鼠 / 关键时刻靠谱
- **推荐干员**: 蜂医 / 蛊

> 你的游戏哲学很简单：搜就完了。据点？守着呗，顺便把周围三个房间翻个底朝天。你对物资的嗅觉比对敌人的嗅觉灵敏十倍，队友在前面打生打死，你在后面默默把每个箱子舔得干干净净。但神奇的是你总能在关键时刻掏出一个谁都没有的金装备。

---

### DLCS - 冷血的蝎

- **关键词**: 幽灵狙击 / 极致耐心 / 一击致命 / 暗处死神
- **推荐干员**: 露娜 / 骇爪

> 你趴在谁都想不到的角落，瞄准镜里是三百米外毫无察觉的目标。一枪，倒。换位置，继续。你是战场上看不见的死神，像渡鸦一样冷酷优雅。你不喜欢近战不是因为打不过，是因为那样不够干净。距离就是你的信仰，耐心就是你的武器。

---

### DLCI - 害群的马

- **关键词**: 不可预测 / 鬼才走位 / 既秀且坑 / 战场变数
- **推荐干员**: 乌鲁鲁 / 深蓝

> 你是对面的噩梦，偶尔也是队友的噩梦。你的蹲点位置鬼才般不可预测，你的行动逻辑连AI都算不出来。有时候你单杀了对面三个人力挽狂澜，有时候你在自己家门口被自己的陷阱炸死。你就像典狱长一样不讲道理——区别是典狱长赢了，你不一定。

---

### DLRS - 怯战的鼠

- **关键词**: 生存大师 / 地图学者 / 极致苟命 / 笑到最后
- **推荐干员**: 骇爪 / 比特

> 你是活到最后的人。不是因为你能打，是因为你根本不打。你把「苟」发展成了一门学问，哪里有人你就往反方向跑，哪里安全你就往那里钻。你研究地图的热情比打枪高十倍，每条阴沟、每个暗道你都门儿清。虽然队友经常问「你人呢」，但你总是笑到最后的那个。

---

### DLRI - 摸鱼的仓鼠

- **关键词**: 咸鱼本鱼 / 舔包达人 / 躺平至上 / 你们打我看
- **推荐干员**: 随缘选人

> 你打游戏的核心诉求就是——别累着。找个安全角落搜搜东西，看看队友打架的直播，时不时捡个漏。枪声一响你就蹲下，枪声一停你就出来舔包。你是三角洲行动的「躺平学」代言人，战绩单上的数据对你来说只是装饰。你不理解为什么有人打游戏还这么累。

## 题目设计说明

共 16 道题目，覆盖四个维度：

| 维度 | 主测题数 | 题目编号 |
|------|----------|----------|
| T/L (团队/独狼) | 6 题 | Q1, Q2, Q3, Q8, Q14, Q15 |
| A/D (猛攻/防守) | 3 题 | Q4, Q11, Q16 |
| C/R (战斗/资源) | 4 题 | Q5, Q6, Q7, Q12 |
| S/I (策略/直觉) | 3 题 | Q9, Q10, Q13 |

每道题的选项同时可能影响多个维度的分数，正分偏向左极 (A/T/C/S)，负分偏向右极 (D/L/R/I)。最终各维度得分 >= 0 取左极字母，< 0 取右极字母，四个字母拼成人格代码。
